About Settings

Data Storage: All timestamps are stored in UTC (Coordinated Universal Time) in the database. This ensures consistency regardless of server timezones or timezone changes.

Display Conversion: Timestamps are automatically converted from UTC to your selected display timezone when shown in the UI, API responses, and charts.

Changing Timezone: You can change the timezone at any time. All existing and new timestamps will immediately display in the new timezone. No data migration is required.

Language: Currently only English is supported. Additional languages will be available in future updates.
